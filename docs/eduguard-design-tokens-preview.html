6. Do's & Don'ts

Nên làm

  • Một primary CTA (tertiary) mỗi section
  • Link text #1E40AF, button #1D4ED8
  • Elevation bằng neutral / surface / border
  • Semantic + icon + text cho mọi badge
  • Negative space — neutral là feature

Không làm

  • Gradient theatrical (Citrus, Indigo, Blush)
  • Box-shadow trên card/container
  • tertiary cho link inline hoặc accent thứ hai
  • Hardcode hex trong component — dùng CSS var
  • Màu swatch trang trí làm UI chrome
